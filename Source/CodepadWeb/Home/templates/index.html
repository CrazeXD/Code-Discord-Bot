Codepad

A Discord Bot Made For Coders.

Features:

  • Code Directly On Discord or from your web browser!
  • Set up cool challenges for your server to participate in!
  • Store up to 500MB of code in the cloud!
  • Get access to features such as IntelliSense and Syntax Highlighting.
  • Run code directly from your browser!



  • Click Me to Invite The Bot!

How It Works

The Discord Bot uses embeds from our code editor website to let you write and run code directly from Discord. The inbuilt text editor gives you access to features like IntelliSense and syntax highlighting for the supported languages.

You can make an account to save and access your codes all from the cloud.

Admins are able to set up highly customizable events as challenges for the server.
